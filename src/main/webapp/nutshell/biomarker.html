In a nutshell

  1. Select the level of interest (e.g. Genus)
  2. Select a statistical test
  3. Press Select Mode
  4. Press Run Analysis
  5. The predictive power of each taxon to discriminate between the two classes is evaluated by Area Under the Curve (AUC). Additionally, the p-value of the selected test, the odds ratio (with upper and lower confidence intervals), delta and fold change are given.
