In a nutshell

  1. Select a rank (e.g. OTU)
  2. Select multivariate method (e.g. PcOA) and press "Select Mode"
  3. Press "Draw Chart".
  4. To assess associations between environmental variables and community composition, set Type to PCA+, Heatmap+, RDA+ or CCA+ and press "Select Mode" followed by "Draw Chart".
