In a nutshell

  1. Select your rank of interest (e.g. OTU).
  2. Select the plot Type (e.g. Boxplot, Barchart or AbundancePlot).
  3. Select the diversity index (e.g. Shannon, Richness, or Evenness) and press "Draw Chart".
  4. The AbundancePlot displays for each sample the number of reads that were assigned to each taxon/OTU of the selected rank as a boxplot.
