In a nutshell

  1. Select a rank (e.g. Family)
  2. To view sign. differently abundant bacterial groups, set Type to AnovaPlot and press "Select Mode".
  3. Press "DrawChart"
  4. Choose a Color scheme according to your preferences.
  5. Select the width, height and resolution of the figure and press "Draw Chart" again. Increase the figure width and height if labels overlap, labels are displayed only partially, or the legend overlaps with the chart.
  6. Press "DrawChart"
